# simple-booking-manager
A basic booking management system with minimal features.

## Tech Stack
- Node.js
- Express.js
- TypeScript
- MongoDB
- Mongoose
- Axios
- React
- Vite
- Tailwind CSS

## Setup

1. Clone the repository
2. Install dependencies
3. Set up environment variables
4. Run the server
5. Run the client

### Features
- Create booking
- View bookings
- Basic validation
